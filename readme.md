# Books Distribution Expense Tracker (Tkinter + SQLite)

## Setup (VS Code)
1. Clone or copy this project folder to your machine.
2. Create & activate a Python venv (recommended):
   - Windows:
     ```
     python -m venv .venv
     .venv\Scripts\activate
     ```
   - macOS/Linux:
     ```
     python -m venv .venv
     source .venv/bin/activate
     ```
